# CIS Benchmark 점검 시나리오

## 개요
CIS Benchmark 기준을 기반으로 Apache/IIS 보안 점검 항목을 자동화하고, 반복 가능한 점검 시나리오로 구성한 문서입니다.

## 목표
- 서버 보안 기준 항목 자동 점검
- 점검 결과 리포트 자동 생성
- 실무 점검 시나리오의 재사용성 확보

## 점검 흐름
- **Target 범위 정의**: Apache/IIS 구성 항목
- **점검 항목 자동화**: 기준별 설정 검증
- **결과 수집/정리**: 로그 및 결과 요약
- **리포트 출력**: 점검 결과 보고서 생성

## MITRE ATT&CK 매핑 (요약)
이 시나리오는 **방어 점검 자동화** 목적이며, 공격 TTP 매핑은 직접적이지 않습니다.

## 시뮬레이션 관점 포인트
- 점검 자동화 스크립트/바이너리의 배포 용이성
- 기준 변경 시 규칙 업데이트 절차
- 결과 보고서의 재현성 및 감사 대응
